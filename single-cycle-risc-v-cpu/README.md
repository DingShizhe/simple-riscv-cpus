# single-cycle-risc-v-cpu

### 慢慢来不着急

###  支持的 risc-v 指令有：

| 指令类型 | I - type | S - type | R - type | B - type | J - type | U - type |
| ------- |----------|----------|-----------|-----------|---------|---------|
| **指令** | **addi** |  **sw**  | **add**  | **beq**   | **jal**  | **lui** |
|          | **lw**   |	        | **sub**   |  **bne**  |	       |	       |     
|          | **slli** |    		|    		|  **blt(u)**	|		|	         |
|          | **jalr** |			|	    	|  **bge(u)**	|		|	         | |



### 宏指令及其对应的原指令

|指令|**addi**|**jalr**|**blt(u)**|**bge(u)**|**jal**|
|-----|-----|-----|------|-------|-------|
|**宏指令**|**li**|**ret**|**bgt(u)**|**ble(u)**|**j**|
|		|**nop**|**jr**|		|		|		|
|		|**mv**|		|		|		|		| |
